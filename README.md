# flowly
Flowly é um PWA de luxo para gestão de agendamentos. Conta com agenda inteligente em visões diária e semanal, seletor de dias estilo 'pílula' com pontos coloridos por status e suporte a múltiplos serviços por cliente. Inclui biblioteca de modelos para WhatsApp editáveis com variáveis, facilitando o envio de lembretes profissionais.
